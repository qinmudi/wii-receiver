wii-receiver
========

代码部署接收端

# 安装使用/npm

## 安装
```
npm i wii-receiver -g
```

# 启动服务
```
wii-receiver        //默认端口为8999
wii-receiver 8888   //修改端口为8888
```

# 安装使用/源代码

## 拉取代码
```
git clone https://github.com/qinmudi/wii-receiver.git
```

## 启动服务
```
node server.js          //默认端口为8999
node server.js 8888     //修改端口为8888
```
